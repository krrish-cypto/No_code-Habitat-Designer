Interactive Space Habitat Designer
A Project by Team No_Code

Welcome to our Interactive Space Habitat Designer! This is a web-based 3D prototyping tool we built to make designing habitats for space missions accessible, intuitive, and fun. Whether you're a student learning about space exploration, an enthusiast dreaming of life on Mars, or an engineer doing preliminary layout work, this tool is for you.

Our goal was to create an application that combines ease of use with powerful design and analysis features, allowing you to quickly visualize and validate your habitat concepts based on real-world constraints.

✨ Core Features
This prototype is packed with features to give you a comprehensive design experience:

Full 3D Environment: Place, move, rotate, and scale objects in a real-time 3D viewport.

Mission Configuration: Set key parameters like Destination (Mars, Moon, LEO), Crew Size, and Mission Duration to define your design constraints.

Customizable Habitats: Start your design by creating a basic habitat shell, choosing from Cylinder, Box, or Dome shapes with custom dimensions.

Rich Object Library: Populate your habitat from a library of essential zones and objects, including sleeping quarters, galleys, labs, gym equipment, and even astronaut models for scale.

Real-Time Analysis: Get instant feedback on your design! The analysis panel checks your layout against NASA-inspired guidelines for volume per crew member and proper zoning (e.g., keeping noisy areas away from sleep stations).

Intelligent Auto-Layout: Stumped for ideas? Let our "Generate Layout" feature create a logical starting point for you based on your mission parameters.

Full User Control: With Undo/Redo, multiple Camera Modes (3D Orbit and Top-Down), and Access Path Drawing, you have precise control over every aspect of your design.

Save Your Work:

Cloud Saves: Sign up for an account to save your designs to the cloud and access them from anywhere.

Guest Mode: Continue as a guest to save your progress directly in your browser's local storage.

🚀 How to Get Started
Getting started is simple:

Open index.html: Just open the file in a modern web browser (like Chrome, Firefox, or Edge).

Login or Continue as Guest: You can create an account to save your work online or just jump right in as a guest.

Set Your Mission: Use the sidebar to define your mission parameters.

Create the Habitat Shell: Choose a shape, set the dimensions, and click "Create Habitat."

Start Designing! Use the object library to add items. Select them to move, rotate, or scale them into the perfect position. Keep an eye on the Analysis panel for feedback.

Save Your Masterpiece: Click the save icon to store your design for later.

🛠️ Tech Stack
We built this project using a completely client-side stack, which means it runs entirely in your browser:

HTML, CSS, JavaScript: The foundation of the web.

Tailwind CSS: For styling the user interface.

Three.js: The magic behind the 3D rendering and interaction.

Firebase: Powers our user authentication and cloud database (Firestore) for saving designs.
